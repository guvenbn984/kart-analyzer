# Kart Analyzer — App UI Kit

## Overview
Hi-fidelity interactive recreation of the Kart Race Analyzer web app.
Based directly on the source `index.html` from `guvenbn984/kart-analyzer@main`.

## Structure
| File | Description |
|---|---|
| `index.html` | Full interactive prototype — upload screen → analysis app |
| `Components.jsx` | Shared primitives: Card, SummaryCard, EventInfo, Badge |
| `Header.jsx` | App header with gradient title + back button |
| `UploadScreen.jsx` | Upload zones, session config, history list |
| `AnalysisApp.jsx` | Tabbed analysis view with standings, pace, driver cards |

## Design Width
1200px desktop. Responsive to 768px.

## Fonts
System stack: `'Segoe UI', system-ui, -apple-system, sans-serif`
No webfonts required.
